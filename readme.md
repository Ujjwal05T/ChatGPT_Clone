## Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/chatgpt-clone.git
cd chatgpt-clone
```
 2. Install dependencies:

 ```bash
npm install
```
3. Create a `.env` file:
```
VITE_API_URL=your_api_url_here
```

4. Start the development server:
```
npm run dev
```
